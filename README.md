`Mockturtle Parser` parses the config file for `Mockturtle`.

## Install

### Swift Package Manager

```swift
dependencies: [
    ...,
    .package(url: "https://github.com/thepeaklab/mockturtle-parser.git", from: "0.1.0")
]
```
